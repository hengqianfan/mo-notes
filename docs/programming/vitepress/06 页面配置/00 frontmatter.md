# frontmatter 配置
::: tip 介绍

---

:one:  &nbsp; `vitepress` 把页面级别的配置称为 frontmatter 配置。  

:two:  &nbsp; 默认主题的配置是在 `config` 文件中配置，但`frontmatter配置`是在每个页面中自行配置，也可以不配置。  

:three:  &nbsp; **需要注意的是**：  
>  frontmatter 配置要写在文档的最顶部，用 " --- " 包裹配置的内容。  
:::

> <Badge type='warning'>代码演示</Badge>  
```md
---

<!-- 页面布局 -->
layout: page 

---
```
