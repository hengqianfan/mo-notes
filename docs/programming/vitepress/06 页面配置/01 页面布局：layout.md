页面配置中  `layout`(布局)  

提供三种模式

> `home`   &nbsp; 提供固定的主页样式，以快速构建主页，但是单调  
> `doc`   &nbsp;  &nbsp; 这是所有页面的**默认布局**，自动将默认样式应用在markdown文档中  
> `page`   &nbsp; 去除默认样式，用于需要自定义页面样式的页面，自由度高  

---

::: info  <Badge type='info'>代码演示</Badge>

> `请在文档的最顶部配置`

```md
---

layout: page

--- 
```

:::
