## 配置 node

- 官网 [enter](https://nodejs.org/en){target="_blank"}

- 更换镜像源

```shell
npm config set registry https://registry.npmmirror.com
```
- **开放脚本权限**

> 使用管理员权限的 `shell` 选择 Y 或者 A

```shell
set-ExecutionPolicy RemoteSigned
```

- 必装的全局包

```shell
npm i nodemon -g
```

```shell
npm i pnpm -g
```
