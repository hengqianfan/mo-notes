


##  `emoji` 的使用方法

- **文档中用 `:` 包裹对应的 emoji 的名称，即可输出为 emoji 表情**  
- **代码格式：`:name:`**  

---

**<small><small>防呆提示：请用英文冒号</small></small>**

::: info  <Badge type='info'>代码示例</Badge>
```markdown
:boom:  
:star:  
:heart:
```
:::



::: info  <Badge type='info'>效果演示</Badge>

:boom:  
:star:  
:heart:  

:::
