- 也是分支结构

## if 语句

### 单分支

```javascript
// 条件满足就执行
if(条件){

}
```

## 双分支

```javascript
if(条件一){
    // 代码一
}else{
    // 代码二
}
```

## 三元运算符

> 条件 ？ 满足执行的代码 : 不满足执行的代码

## switch 语句

```javascript
switch (key) {
    case value:
        // 代码一
        break;
    case value:
        // 代码二
        break;
    default:
        // 默认执行的代码
        break;
}
```
