## 进入开源界面

[enter](https://github.com/alist-org/alist/releases){target="_blank"}

选择对应版本

- 大多数人是 `alist-windows-amd64.zip`这个文件

- 解压后会得到一个exe文件

- 在该文件目录下，打开CMD

```shell
alist server
```
- 打开浏览器

- 输入用户名与密码(用户名为admin)
